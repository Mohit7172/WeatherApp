🌦 MK Weather App: Discover & Forecast
A feature-rich VB.NET desktop application that delivers real-time weather updates and 5-day forecasts using the OpenWeatherMap API.
🚀 Features

🔐 User Authentication: Secure login and signup with Microsoft Access backend.
🌍 Real-Time Weather: Search current weather by city name.
📆 5-Day Forecast: Detailed weather predictions using JSON-based OpenWeatherMap API.
🕒 Last City Auto-Load: Automatically loads the last searched city on app launch.
📜 Search History: View, delete, or clear all search history.
❤️ Favorite Cities:
Save, view, and delete favorite cities.
Auto-suggest favorite cities during input.


🎨 Modern UI: Sleek and intuitive WinForms interface built with VB.NET.
🔒 Secure Data Access: Robust ADO.NET implementation for database operations.

🛠 Technologies Used

VB.NET (Visual Studio 2022)
ADO.NET for database connectivity
Microsoft Access 2016 for data storage
OpenWeatherMap API for weather data
WinForms for GUI design


🗄 Database Tables

Users: Stores user login credentials (username, password).
SearchHistory: Logs each user’s weather search history.
FavoriteCities: Stores user-specific favorite cities for quick access.
